JIT Shooter
http://github.com/dabroz/dablang-jitshooter

Simple game created with Dab 0.1.

Running:

Easiest way is to copy binary dab.exe into JIT Shooter directory, run it and press F5.

Controls:

A/D -- movement
Space -- fire
